### A

translate3d(x, y, z) 第1个参数对应X轴，第2个参数对应Y轴，第3个参数对应Z轴，参数不允许省略！

```css
/* 3 个参数 */
.box{
    transform: translate3d(100px, 100px, 0);
}
```

指定对象的 3D 旋转角度，其中前3个参数分别表示旋转的方向x,y,z，第4个参数表示旋转的角度，参数不允许省略

```css
/* 4 个参数 */
.box{
    transform: rotate3d(0, 0, 1, 45deg);
}
```
